# Tabbed Sections

A tabbed section is defined by adding the class `tabs` to a Markdown Heading.  Each sub section then becomes a tab.  For instance,

```juncture
## Tabbed Section Example
{: .tabs}

### Tab 1

Tab 1 content...

### Tab 2

Tab 2 content...

### Tab 3

Tab 3 content...
```
